# Truebase Dashboard

React + Vite + Tailwind dashboard implemented from the [Wireframe directives](https://www.figma.com/design/wL8l4TpqLiF4UN8tu10zGE/Wireframe-directives?node-id=706-7889) Figma design.

## Run locally

```bash
npm install
npm run dev
```

Open [http://localhost:5173](http://localhost:5173).

## Deploy (GitHub Pages)

1. **Enable GitHub Pages:** In your repo go to **Settings → Pages**.
2. Under **Build and deployment**, set **Source** to **GitHub Actions**.
3. Each push to `main` will build and deploy. The site will be at:
   **https://saldrao-77.github.io/3-2-claude-code-proj/**

Builds run from `.github/workflows/deploy.yml`.
