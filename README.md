# Truebase Dashboard

React + Vite + Tailwind dashboard implemented from the [Wireframe directives](https://www.figma.com/design/wL8l4TpqLiF4UN8tu10zGE/Wireframe-directives?node-id=706-7889) Figma design.

## Run locally

```bash
npm install
npm run dev
```

Open [http://localhost:5173](http://localhost:5173).

## Deploy (Vercel)

With Vercel connected to this GitHub repo, pushes to `main` auto-deploy. Vercel detects the Vite app and uses `npm run build`; the app is served from the `dist` output.
